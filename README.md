easyreader
==========

Custom JS/CSS to make Wikipedia pages more comfortable to read. If you have a registered Wikipedia account, you can use this by copying it to your custom JS/CSS files. Please note that this will only work with the default Vector skinâ€”it will need serious tweaking for any other skins.

##To install
1. Copy the contents of the JS file to your [common.js](https://en.wikipedia.org/wiki/Special:MyPage/common.js).
2. Copy the contents of the CSS file to your [common.css](https://en.wikipedia.org/wiki/Special:MyPage/common.css).
3. [Bypass your browser's cache](https://en.wikipedia.org/wiki/Wikipedia:Bypass_your_cache).
